# Color Picker

這是一個簡單的顏色選擇器應用程式，使用者可以通過點擊來選擇螢幕上的顏色，並顯示與該顏色最接近的 Minecraft 方塊。

## 安裝

1. 克隆此專案到本地端：
    ```bash
    git clone https://github.com/yourusername/colorpick.git
    ```
2. 進入專案目錄：
    ```bash
    cd colorpick
    ```
3. 安裝所需的依賴：
    ```bash
    npm install
    ```

## 使用

1. 啟動應用程式：
    ```bash
    npm start
    ```
2. 在瀏覽器中打開 `index.html` 文件。
3. 移動滑鼠以顯示顏色選擇器，點擊以選擇顏色。
4. 選擇的顏色將顯示在螢幕上，並列出與該顏色最接近的 Minecraft 方塊。
